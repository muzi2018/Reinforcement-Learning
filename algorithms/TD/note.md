| 特点         | Monte Carlo         | Temporal Difference    |
| ------------ | ------------------- | ---------------------- |
| 更新时机     | 等完整 episode 结束 | 每一步就更新           |
| 是否需要模型 | 不需要              | 不需要（model-free）   |
| 收敛速度     | 稳定，但慢          | 可能更快，收敛逐步逼近 |
| 样本利用     | 每条路径只用一次    | 每一步都能利用样本     |
